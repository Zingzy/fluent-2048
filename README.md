# Fluent 2048

A Fluent and modern Version of the classic 2048 Game (with mica)

**Works with windows 11 only**

## Features

- Modern Design with Mica
- Reset Button
- HighScore tracking
- Pin Mode
- Custom Backdrop Materials and window styles
